ProblemOnlyD Class Reference

#include <panoc-alm/util/problem.hpp>

Detailed Description

Moves the state constraints in the set C to the set D, resulting in an unconstraint inner problem.

The new constraints function g becomes the concatenation of the original g function and the identity function. The new set D is the cartesian product of the original D Ã— C.
